## 백준 11725 트리의 부모 찾기

문제 링크 : [트리의 부모 찾기](https://www.acmicpc.net/problem/11725)

### 문제 조건

루트 없는 트리가 주어진다. 이때, 트리의 루트를 1이라고 정했을 때, 각 노드의 부모를 구하는 프로그램을 작성하시오.

#### 입력

- 노드의 개수 N (2<=N<=100,000)

#### 출력

- 첫째 줄부터 N-1개의 줄에 각 노드의 부모 노드 번호를 2번 노드부터 순서대로 출력한다.

### 문제 접근

- 각 입력에 대해 양방향 그래프 만들기
    - 각 노드에 연결되어 있는 노드들을 기록하기 위해 Array(N){ MutableList\<Int> } 로 관리
- 루트노드부터 순차적으로 bfs 탐색 + 이미 지나간 노드인지 판단하기 위한 BooleanArray
